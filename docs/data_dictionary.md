# Data Dictionary

-Text: The original text as imported from the source before any processing.

-Label: The classification that determines whether the text is "human-generated" (written by a human) or "AI-generated" (generated by artificial intelligence).

-Words: A list of the words that make up the text after being broken down into separate words.

-Word Length Frequency: A distribution showing the frequency of each word length in the text (e.g., the number of words with a length of 3 letters, 4 letters, etc.).

-Paragraphs: The number of paragraphs in each text, i.e., dividing the text into paragraphs and counting the number of paragraphs.

-Sentences: The number of sentences in each text, i.e., dividing the text into sentences and counting the number of sentences.

-Total Number Sentences: The total number of sentences in a single text.

-Total Number Paragraphs: The total number of paragraphs in a single text. avg_sentences_per_paragraph: Average number of sentences per paragraph.

-num_words_in_top50_embeddin: Number of words in the text that fall within the top 50 most important words in embedding representations.

-perplexity: A measure of how difficult a text is for a language model; a higher number indicates a less predictable text.

-gpt_output_probability: A probability assigned by the GPT model to the text, representing the model's confidence that the text meets its predictions.

-cleaned_text: Text after cleaning up unwanted elements and preparing it for use.
